What is HTML ?

HTML is the standard language used to create and structure content on the web.
It defines how elements like text, images, links, and multimedia appear in web browsers.


Full form of HTML 

 H - Hyper 
 T - Text 
 M - Markup 
 L - Language

H for Hyper , T for Text , M for Markup , L for Language

What is Tag ?

In Html, Tag decide the structure of the Content.
