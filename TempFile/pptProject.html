PowerPoint Practical Work Instructions

📘 Topic 1: My Self Introduction Presentation

🎯 Objective: Apne baare mein PowerPoint ke madhyam se introduction dena.

📋 Instructions:

  • Kam se kam 5 Slides hone chahiye.
  • Har slide me text + image hona chahiye.
  • Transition effect use karein.
  • File ko apne naam se save karein: MyIntro_YourName.pptx

🖥️ Suggested Slide Structure:

Slide No.TitleContent
1My IntroductionName, Age, Photo, Place
2My HobbiesDrawing, Dancing, Singing (with images)
3My EducationClass, Subjects, School Name
4My GoalsFuture Career Plans (Doctor, Engineer...)
5My Favourite ThingsFood, Colour, Place, Quote, etc.

🛠️ PowerPoint Tools to Use:

  • Insert > Text Box
  • Insert > Picture
  • Design > Theme
  • Transitions > Apply to All

🎉 Topic 2: Event Invitation Presentation

🎯 Objective: PowerPoint ka use karke ek digital invitation banana.

📋 Instructions:

  • 5 slides banana hai kisi event ka invitation create karne ke liye.
  • Har slide me images aur stylish text use karein.
  • File save karein: EventInvite_YourName.pptx

🖥️ Suggested Slide Structure:

Slide No.TitleContent
1You're Invited!Event name, date image or logo
2Event DetailsWhat, When, Where (Date, Time, Venue)
3About the HostNames, Message, Theme/Quote
4Special ProgramsDance, DJ, Cake Cutting, etc.
5Thank You & RSVPContact info, See you soon!

🛠️ Tools to Use:

  • Fancy Fonts using WordArt
  • Design Themes
  • Transitions & Animations
  • Background Music (optional)

📢 Topic 3: Product Advertisement Presentation

🎯 Objective: Apne imaginary ya real product ka ad presentation banana.

📋 Instructions:

  • Product ka naam aur tagline create karein.
  • Use benefits, price, images batayein.
  • Presentation me style aur animation ka use karein.
  • File name: ProductAd_YourName.pptx

🖥️ Suggested Slide Structure:

Slide No.TitleContent
1Product Name & TaglineLogo, tagline, catchy intro
2FeaturesWhat makes your product unique
3Price & OfferPrice, discount, limited offer
4Why Choose Us?Comparison with other brands
5Contact & Buy NowWebsite, contact, closing line

🛠️ Tools to Use:

  • Insert Icons / Shapes
  • SmartArt for comparison
  • Transitions + Animations
  • Custom Background

🌍 Topic 4: Educational Topic – Solar System

🎯 Objective: Ek educational topic par presentation banana jaise "Solar System", "Water Cycle", etc.

📋 Instructions:

  • Choose any science or educational topic.
  • Content samajhne layak aur visual hona chahiye.
  • Slide me text + diagram/image ka combination ho.
  • File Name: EduTopic_YourName.pptx

🖥️ Suggested Slide Structure (Solar System):

Slide No.TitleContent
1IntroductionWhat is Solar System?
2The SunMain source of energy
3PlanetsAll 8 planets in order
4Interesting FactsAny 3–4 surprising facts
5ConclusionSummary + image or video

🛠️ Tools to Use:

  • Insert > Picture / Diagram
  • Animations for movement
  • SmartArt for sequence
  • Design + Transitions
